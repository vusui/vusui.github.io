列表

定义<ul>、<ol>、<dl>标签样式。

无序列表

  • 使用.vus-ul类，作为<ul>标签样式。
  • 可添加的类有：
  • .disc：实心圆点。
  • .circle：空心圆点。
  • .square：实心正方形。
实心圆点
  • 关关雎鸠，在河之洲。
  • 窈窕淑女，君子好逑。
空心圆点
  • 参差荇菜，左右流之。
  • 窈窕淑女，寤寐求之。
实心正方形
  • 求之不得，寤寐思服。
  • 悠哉悠哉，辗转反侧。
混合
  • 参差荇菜，左右采之。
    • 窈窕淑女，琴瑟友之。
  • 参差荇菜，左右芼之。
    • 窈窕淑女，钟鼓乐之。
<!-- 实心圆点 -->
<ul class="vus-ul disc">
   <li>关关雎鸠，在河之洲。</li>
   ...
</ul>

<!-- 空心圆点 -->
<ul class="vus-ul circle">
   <li>参差荇菜，左右流之。</li>
   ...
</ul>

<!-- 实心正方形 -->
<ul class="vus-ul square">
   <li>求之不得，寤寐思服。</li>
   ...
</ul>

<!-- 混合 -->
<ul class="vus-ul square">
   <li>
      参差荇菜，左右采之。
      <ul class="vus-ul disc">
         <li>窈窕淑女，琴瑟友之。</li>
      </ul>
   </li>
   <li>
      参差荇菜，左右芼之。
      <ul class="vus-ul circle">
         <li>窈窕淑女，钟鼓乐之。</li>
      </ul>
   </li>
</ul>

有序列表

  • 使用.vus-ol类，作为<ol>标签样式。
  • 可添加的类有：
  • .number：常规数字。
  • .decimal：十进制数。
  • .lower-roman：小写罗马文字。
  • .upper-roman：大写罗马文字。
  • .lower-latin：小写拉丁文。
  • .upper-latin：大写拉丁文。
常规数字
  1. 当你老了，
  2. 头发花白，
  3. 睡意沉沉，
  4. 倦坐在炉边，
十进制数
  1. 取下这本书来，
  2. 慢慢读着，
  3. 追梦当年的眼神，
  4. 你那柔美的神采与深幽的晕影。
小写罗马文字
  1. 多少人爱过你昙花一现的身影，
  2. 爱过你的美貌，
  3. 以虚伪或真情，
  4. 惟独一人曾爱你那朝圣者的心，
大写罗马文字
  1. 爱过你的美貌，
  2. 以虚伪或真情，
  3. 惟独一人曾爱你那朝圣者的心，
  4. 爱过你的美貌，
小写拉丁文
  1. 以虚伪或真情，
  2. 惟独一人曾爱你那朝圣者的心，
  3. 爱你哀戚的脸上岁月的留痕。
  4. 在炉罩边低眉弯腰，
大写拉丁文
  1. 忧戚沉思，
  2. 喃喃而语，
  3. 爱情是怎样逝去，
  4. 又怎样步上群山，
  5. 怎样在繁星之间藏住了脸。

桃花坞裏桃花庵，桃花庵裏桃花仙。
桃花仙人种桃树，又折花枝当酒钱。
酒醒只在花前坐，酒醉还须花下眠。
花前花後日复日，酒醉酒醒年复年。
不愿鞠躬车马前，但愿老死花酒间。
车尘马足贵者趣，酒盏花枝贫者缘。
若将富贵比贫贱，一在平地一在天。
若将贫贱比车马，他得驱驰我得闲。
世人笑我忒风颠，我咲世人看不穿。
记得五陵豪杰墓，无酒无花锄作田。
